About This Project

The Django Enquiry System is a production-ready backend application designed to demonstrate robust form handling and real-time notification integrations.

Use Case

In many business websites, "Contact Us" forms are often the primary lead generation tool. This project solves the problem of unreliable communication by ensuring that every enquiry is validated, stored securely, and immediately dispatched to the relevant stakeholders via multiple channels (Email and WhatsApp).

Technical Stack

  • Django 5 & Python: The core framework providing the MVC architecture and ORM.
  • PostgreSQL: Production-grade relational database for data persistence.
  • SMTP Integration: Secure email dispatch using Gmail's SMTP server.
  • Twilio API: Real-time WhatsApp messaging integration via a service-layer architecture.

Engineering Highlights

This project focuses on clean architecture and reliability. It features rigorous environment variable management for security, a dedicated service layer for third-party APIs to keep views clean, and graceful error handling to ensure a smooth user experience even if external services experience downtime.
